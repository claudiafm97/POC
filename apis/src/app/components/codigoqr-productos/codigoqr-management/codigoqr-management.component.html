Código QR

La API de Gestionar QR permite la generación del código QR y la notificación de la transferencia exitosa. Para entender cómo funciona está API, solo debes realizar los siguientes 4 simples pasos, ingresar los datos del producto que se va a comprar, luego los datos de tu tienda o negocio, después puedes generar el código y finalmente recibir una notificación de confirmación de la tranferencia exitosa.

Cabe resaltar que esta prueba de concepto es una muestra representativa de las funcionalidades de está API.
