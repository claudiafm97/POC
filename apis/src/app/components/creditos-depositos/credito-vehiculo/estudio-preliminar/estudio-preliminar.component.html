Familias de Segundo Orden

Descripción

Aqui encontrarás las Familias de API's de segundo orden de la familia Crédito de Vehiculo, las cuales te servirán para realizar un estudio completo para otorgar un crédito de Vehiculo Sufi. Para completar el estudio de crédito de vehiculo, es necesario completar correctamente los estudios:
  1. Estudio Preliminar
  2. Estudio Final
Si ocurre algún inconveniente en el proceso. Diríjase a consumir el API de Estudio Manual y realice el proceso pertinente

Estudio Preliminar

Capacidad que permite el registro de la información para realizar el estudio preliminar para el crédito para véhiculo, registrando los datos básicos del cliente y retornando un identificador único necesario para la radicación de toda la información del estudio del crédito


Last updated 3 mins ago

Estudio Final

Capacidad que permite la radicación de la información adicional necesaria para realizar el estudio final del crédito para véhiculo, registrando los datos personales adicionales del solicitante del crédito y teniendo como enlace a los datos previos el identificador unico entregado por medio del registro de los datos basicos del cliente.


Last updated 3 mins ago

Estudio Manual

Capacidad que permite la radicación de la información necesaria para realizar el estudio de crédito nanual para véhiculo, registrando los datos suplementarios del cliente


Last updated 3 mins ago
