Consulta de información de documento

El producto Consulta de información de documento, cuenta con dos modalidades: desde la perspectiva de un usuario realizando su proceso de apertura de una cuenta de ahorros via virtual y la otra es a partir de una sucursal física de bancolombia. Si deseas conocer el paso a paso solo debes oprimir en "Comenzar".

Características del producto

    Verificación rápida y transparente

    Disminución del fraude

    Accesibilidad a información para consulta
